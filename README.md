Клиент-серверное приложенное, реализованное на Python. 
Пользователь вводит количество котов в форме, сервер обрабатывает запрос и возвращает страницу с указанным числом изображений кота.

Для запуска приложения:
1. Установить зависимость:
  pip install flask
2. Запустить приложение:
  python app.py
3. Открыть в браузере:
  http://127.0.0.1:5000
