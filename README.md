Клиент-серверное приложенное, реализованное на Python. 
Пользователь вводит количество котов в форме, сервер обрабатывает запрос и возвращает страницу с указанным числом изображений кота.

Для запуска приложения:
1. Установить зависимость:
  pip install flask
2. Запустить приложение:
  python app.py
3. Открыть в браузере:
  http://127.0.0.1:5000

p2p:
Консольное приложение на python для обмена сообщениями с помощью сокетов по прицнипу p2p (каждый участник может работать как клиент и как сервер).
Для запуска приложения:
1. Зайти в папку p2p:
   cd p2p
3. Прописать в программе IP-адрес вашего собеседника из локальной сети
4. Запустить программу:
   python app.py
5. То же самое повторить на ПК собеседника (он прописывает ВАШ ip-адрес)
